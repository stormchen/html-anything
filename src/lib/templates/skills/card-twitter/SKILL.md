【模板: Twitter 分享卡】
- 容器 `w-[1600px] h-[900px]`, 暗色 / 亮色二選一根據內容情绪。
- 中央一句 hero 金句 (text-6xl, font-semibold, 限 2-3 行)。
- 下方作者署名 + 頭像占位 + handle。
- 左上角小標籤 (型別: "Insight" / "Data" / "Quote")。
- 右下角品牌水印。
- 整張卡片有微妙的纹理 (grid 網格 / noise / dot pattern)。
- 截圖後可直接配推文發出, 視覺简洁有力。
