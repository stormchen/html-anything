# 5 個讓你的 AI 編輯用得更順手的小习惯

## 封面
建議收藏 · AI 工具黨 必看

## 卡 1: 放弃完美 prompt
你寫的 prompt 不可能一次完美。把它当**對話**, 不是 SQL。

## 卡 2: 讓 AI 看見你的格式
贴一段 markdown / CSV 之前, 先告诉它"我希望輸出像 X 風格"——
給一個具體参考, 比給 50 個形容词管用。

## 卡 3: 用模板, 别每次重寫
把好用的提示词存成模板。HTML Anything 直接選模板就行,
9 套世界级設計 system 現成的。

## 卡 4: 二次編輯跑 diff
重新生成不是从 0 開始。改完內容只發 diff, 讓 AI 在已有 HTML 上微調,
省 80% token + 保持設計風格。

## 卡 5: 复用本地 session
不用 API Key, 讓本地 Claude / Cursor / Codex 跑。0 邊際成本,
速度比遠程 API 快得多。

## 總結
工具不是越多越好, 流程順才是真的好用。
追蹤我, 每周更新 AI workflow 實战。
