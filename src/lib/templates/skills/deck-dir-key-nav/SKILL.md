【模板: 極簡方向键 Keynote】
【意圖】“有話要說但没什么可看” 的極簡 keynote。
【布局】
- 頁數由【用户內容】决定 (短內容 8 頁起步, 長內容應更多); 每頁單色背景, 从下列調色板裡循环選取 (靛 / 奶 / 绛 / 翠 / 灰 / 紫 / 白 / 炭), 同色可复用
- 160px display 標題 + 4px 短粗 accent 线
- 箭頭 → 前缀的 Mono 列表
- 左下 ← → kbd 提示 + 右下頁码
