【模板: Replit Slides Style Deck】
【意圖】Replit Slides 風的單文件 horizontal-swipe deck, 選 1 套佈景主題不混用。
【布局】
- Pick one theme: helix / holm / vance / bevel / world-dark / world-mint / atlas / bluehouse
- Cover + agenda + N 個 content + 收尾 (N 由【用户內容】長度决定, 完整覆盖每個要點; 短內容 6-10 起步, 長內容應更多)
【設計細節】
- 每套佈景主題有完整調色板 + 字體 + accent, 不要混用
