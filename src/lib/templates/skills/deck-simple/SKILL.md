【模板: Simple Deck】
【意圖】干净通用的 horizontal-swipe deck (pitch / overview / study)。
【布局】
- Cover + N 個 content 頁 + 收尾 (N 由【用户內容】長度决定, 完整覆盖每個要點; 短內容 6-10 起步, 長內容應更多)
- 每頁一個核心資訊 + 1 張圖 / 1 個圖表
- 頂部 progress bar
【設計細節】
- 键盘 ← / → 切换 + hash 同步
