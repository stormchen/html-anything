# 新用户上手流程 · HTML Anything

> 白板風便利贴流程圖, 6 節點, 1 個分支

## 流程
1. **打開 localhost:3456** — 浏覽器進入主界面
2. **自動識别本地 agent** — 扫描 PATH, 列出已登入的 CLI
3. **選模板** — 75 個模板按場景分類別
4. **貼上內容 / 上傳文件** — Markdown / CSV / Excel 都支援
5. **⌘+Enter 轉换** — 流式調用 agent → SSE → 即時重新整理預覽
6. **一键複製 / 下載** — 公眾號 / 推特 / 知乎 / PNG / HTML

分支: 若无本地 agent → 引導安装 (Claude / Cursor / Codex)
