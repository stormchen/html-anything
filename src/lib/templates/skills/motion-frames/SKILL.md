【模板: Motion 帧 / Hero Loop】
【意圖】一帧带循环動效的 hero, 可作為影片片頭或落地頁大圖。
【布局】
- Rotating type ring (SVG + transform)
- Animated globe / 抽象几何
- Ticking timer (mono 字體, JS 可有可无)
- Parallax labels 浮動
【設計細節】
- 纯 CSS 動效, 流畅可循环
- 電影感調色 + 1 個霓虹 accent
