# HTML Anything · 產品介绍

## 我们解决的問題
- Markdown 易寫但發出去丑
- 截圖發推丑得離谱
- 公眾號 / 知乎 各家排版规则不一

## 我们的方案
**任何輸入 → 世界级 HTML**, 用你本地的 AI agent

## 三個核心
1. 复用你已經登入的本地 agent (Claude / Cursor / Codex), 不要 API Key
2. 9 套世界级模板 (PPT / 履歷 / 海報 / 小红書 / 數據報告 ...)
3. 一键複製到公眾號 / 推特 / 知乎, 或匯出 PNG

## 數據
- 80 秒生成 31KB 的可可交付文章
- 9 套模板, 8 個本地 agent 適配
- 0 API Key, 100% 复用本地 session

## 為什么是現在
Claude 团队不再用 markdown, 全换成 HTML 了。HTML 是面向讀者的最终形態, AI 讓"囉嗦"不再是問題。

`pnpm dev` → 打開浏覽器 → ⌘+Enter
