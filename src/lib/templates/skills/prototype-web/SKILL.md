【模板: Web 產品原型】
- 輸出一個完整的產品 landing page。
- Sections: Top Nav (logo + 導航 + CTA 按钮) → Hero (大標題 + 副標 + 双 CTA + 可視化占位) → Features (3-6 個特性卡) → How it works (步驟) → Social proof (logo wall / 評價) → Pricing (可選) → Footer。
- 使用現代 SaaS 設計趋势: 大字号、柔和渐变、glassmorphism 卡片、滚動到視圖入場動畫 (pure CSS 即可)。
- 响應式: 行動裝置單欄, 桌面多栏; 至少處理 `md:` 斷點。
- 添加互動: nav 滚動变色; 特性卡 hover 浮起; FAQ 可手風琴展開 (用 `<details>`)。
- 這是高保真原型, 應该讓人覺得"明天就能上线"。
