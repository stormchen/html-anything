【模板: SaaS Landing】
【意圖】完整的 SaaS 產品落地頁, 把用户內容映射到標准 sections。
【布局】
- Top nav (logo + 導航 + sign-in + 主 CTA)
- Hero (大標題 + 副標 + 双 CTA + 可視化占位)
- Logo wall (社會認證)
- Features (3-6 個特性卡, icon + 標題 + 描述)
- How it works (3 步流程, 數字 + 標題 + 描述)
- Pricing (2-3 档, 推荐档高亮)
- FAQ (details/summary 手風琴)
- Footer
【設計細節】
- 現代 SaaS 風: 大字号, 柔和渐变, glassmorphism 卡片, 滚動入場動畫
- 至少處理 `md:` 斷點, 行動裝置單欄
