【模板: 社群媒體矩阵追踪 (Social Matrix Tracker)】
【意圖】電影感、數據密集的多平台社群媒體儀表板。
【布局】
- Hero header (統一品牌 + 時間窗 + 佈景主題切换)
- Multi-platform KPI matrix (横 platform × 纵 metric)
- 互動式 charts (hover tooltip + range compare)
- Insights drawer (右抽屉, 文字洞察)
- Top posts grid
【設計細節】
- dark / light 切换; 數據用 seed 兜底
