【模板: 等候名單頁 / Waitlist】
【意圖】為新產品 / 早鳥內测做一張極簡等候頁。
【布局】
- 居中布局: brand logo + 一行 tagline + 大字 hero (說清楚做什么)
- 郵箱捕获 input + submit 按钮 (合並成一個 pill)
- 下方 3 個小卖點 (icon + 一行字)
- 底部 founders note + 社交連結
【設計細節】
- 装飾: SVG 渐变 mesh / 噪點纹理 / 一顆星轨
- 成功送出後給一個微動效 (✓ + 文案变化)
