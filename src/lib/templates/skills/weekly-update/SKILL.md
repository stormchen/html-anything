【模板: 团队週報 Deck】
【意圖】6-8 頁 horizontal-swipe slides, 週報固定結构。
【布局】
- Cover (周次 + 团队 + 一句話佈景主題)
- Shipped (列表 + owner)
- In flight (進度条)
- Blocked (红色 callout)
- Metrics (KPI 卡片網格 + 周對比)
- Asks (求助清單)
- Thanks 收尾
【設計細節】
- 键盘左右切换, hash 同步
